# Siemens Brandville Quick Start

## 5-Second Slide Creation

### 1. Pick Background
- **Digital/Screen:** Deep Blue `#000028` (default)
- **Print/Accessibility:** Light Sand `#f3f3f0`

### 2. Size Your Title
```
≤ 15 chars  → 80pt
≤ 35 chars  → 60pt (default)
> 35 chars  → 40pt
```

### 3. Choose Text Colors

| Background | Headlines | Body Text |
|------------|-----------|-----------|
| Deep Blue | Bold Green `#00ffb9` | Light Sand `#f3f3f0` |
| Light Sand | Deep Blue `#000028` | Deep Blue `#000028` |

### 4. Select Layout

**Content slides:**
- One message → One Object (large)
- Compare 2 things → Two Columns
- Three pillars → Three Columns
- 2×2 matrix → Four Objects

**Story slides:**
- Opening → Title slide with motif
- Chapter break → Chapter Divider
- Closing insights → Key Takeaways
- Choices → Way Forward
- Decision → Go/No-Go

### 5. Required Elements

Every presentation needs:
- [ ] Key Takeaways slide (3 insights + pros/cons)
- [ ] Quote slide (with source)
- [ ] Contact slide
- [ ] Proper footer on all slides

## Common Patterns

### SCQA Opening
1. Situation slide (what's true)
2. Complication slide (what changed)
3. Question slide (key tension)
4. Answer slide (recommendation)

### Pyramid Principle
- **Title:** Lead with conclusion
- **Body:** 3 supporting arguments
- **Notes:** Evidence and sources

### Footer Template
```
Internal | © Siemens 2026 | Name | Dept | DD.MM.YYYY
```

## Color Quick Pick

**Signature:** `#009999` (Siemens Petrol)
**Dark bg headline:** `#00ffb9` (Bold Green)
**Dark bg body:** `#f3f3f0` (Light Sand)
**Light bg text:** `#000028` (Deep Blue)
**Accent:** `#00e6dc` (Bold Blue)

## Layout Codes Quick Reference

| Code | Content |
|------|---------|
| `ph0` | Title |
| `ph1` | Subtitle / Primary content |
| `ph2` | Body / Right column |
| `ph10` | Footer |
| `ph11` | Slide number / Picture |
| `ph12` | Left column / Picture / Quote |
| `ph13` | Right column / Secondary body |
| `ph14-18` | Extended content areas |

## File Naming

Use descriptive names with version:
```
YYYY-MM-DD_ProjectName_Status.pptx
2026-03-14_DigitalTransformation_Draft.pptx
```
